## Feitiços em Código: Blocos de Código

Um mago moderno também escreve seus feitiços em linguagens de programação. Um simples encantamento para exibir uma mensagem pode ser a função `saudacaoArcana()`. Para magias mais complexas, usamos um bloco de código indentado:

```javascript
/**
 * Um feitiço para encontrar o equilíbrio entre a luz e a escuridão.
 *
 * @param {number} luz A quantidade de energia da luz.
 * @param {number} escuridao A quantidade de energia da escuridão.
 * @returns {string | undefined} Uma mensagem sobre o estado do equilíbrio, ou undefined se o vazio for absoluto.
 */
function encontrarEquilibrio(luz, escuridao) {
  const energiaTotal = luz + escuridao;

  if (energiaTotal === 0) {
    console.log("O Vazio Absoluto. Perigoso, mas poderoso.");
    return;
  }

  const fatorDeLuz = luz / energiaTotal;

  if (fatorDeLuz > 0.4 && fatorDeLuz < 0.6) {
    return "Equilíbrio alcançado. O Nexus está estável.";
  } else {
    return "Cuidado, as forças estão desbalanceadas!";
  }
}
```
