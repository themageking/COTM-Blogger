## O que são Hooks?

Hooks são funções que nos permitem "enganchar" nos recursos de estado e ciclo de vida do React a partir de componentes de função. Antes deles, precisávamos usar classes para gerenciar o estado.

### Principais Hooks

Aqui estão os dois mais famosos:

* **`useState`**: Permite adicionar estado a um componente de função.
* **`useEffect`**: Permite executar efeitos colaterais, como buscar dados em uma API ou manipular o DOM.

Usar Hooks torna nosso código mais limpo e reutilizável.
