## Por que usar Tailwind CSS?

A principal vantagem do Tailwind é que você para de inventar nomes para classes. Em vez de criar uma classe `.card-title`, você compõe o estilo diretamente no HTML.

> A melhor prática é aquela que te faz entregar o projeto no prazo. O Tailwind ajuda nisso.

Isso pode parecer estranho no início, mas depois de um tempo, a produtividade aumenta absurdamente. Você estiliza componentes de forma rápida e consistente.
