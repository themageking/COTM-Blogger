## O que é Ritmo Vertical?

Você já olhou para uma página e sentiu que, mesmo sem saber o porquê, tudo parecia perfeitamente alinhado e agradável aos olhos? A chance de isso ser obra do **ritmo vertical** é enorme.

Ritmo vertical é uma técnica de design que busca criar consistência e harmonia no espaçamento vertical de uma página. A ideia é simples: todos os elementos e seus espaçamentos (tamanhos de fonte, altura de linha, margens, paddings) devem ser múltiplos de uma unidade base comum. Isso cria uma "grade" invisível na qual o texto e os elementos "pousam", resultando em um layout coeso e profissional.

## A Base de Tudo: O `line-height`

O coração do ritmo vertical está no texto. A primeira e mais importante decisão é definir um `line-height` (altura da linha) base para o corpo do seu texto que seja um número fácil de trabalhar.

Um padrão de ouro na web é usar uma fonte base de **`16px`** com um `line-height` de **`24px`**.

Nosso número mágico, nossa unidade base, será **24**.

```css
body {
  font-size: 16px;
  line-height: 24px; /* Nossa unidade base! */
}
